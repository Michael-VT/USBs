## ⚠️ Важные технические детали:

### Treeview vs Listbox:
- **Treeview** использует строковые ID элементов (`"0"`, `"1"`, ...)
- **Listbox** использует числовые индексы (0, 1, 2, ...)
- **identify_row(event.y)** возвращает строковый ID элемента под курсором
- **selection()** возвращает кортеж строковых ID выбранных элементов
- **item(item_id, values=(...))** обновляет значения элемента

### Статусы команд:
- Хранятся в словаре `self.command_status` по индексу команды
- Обновляются через `_update_command_status(cmd_index, status)`
- Статус "✓" для success, "✗" для failed/timeout
- Цвет: зелёный для success, красный для failed

### Window geometry:
- Формат: `"widthxheight+x+y"` (например, "1152x734+5+30")
- Сохраняется в `self.cfg["window_geometry"]`
- Восстанавливается через `self.root.geometry()`
- Сохраняется при выходе, загружается при старте
